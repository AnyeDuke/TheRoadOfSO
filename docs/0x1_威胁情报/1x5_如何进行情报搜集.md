# 如何进行情报搜集？

### 如何收集？

- 拿到数据后，从一个常见的指标进行分析：
  - 网络。IP，URL
  - 主机。文件哈希，动态链接库，注册表
  - 邮件

- 使用开源的威胁情报平台 [TODO]

- 自己搭建蜜罐、蜜网，收集攻击数据（IP，Hash，URL等）

- 对自己的已有资产进行分析建模
